# Evaluación de conceptos básicos de SQL

Contexto

En la base de datos escuela, ahora queremos crear una tabla llamada cursos, donde se registran los cursos disponibles para los alumnos.

# INSTRUCCIONES

1. Crear la tabla cursos

Crea una tabla con la siguiente estructura:

id → INT, PRIMARY KEY, AUTO_INCREMENT

nombre → VARCHAR(50)

creditos → INT

area → VARCHAR(50)

Ejemplo de áreas: Matemáticas, Computación, Humanidades, Administración

2. Insertar al menos 5 cursos

Incluye cursos variados, por ejemplo:

Cálculo Diferencial

Programación I

Literatura Universal

Administración Financiera

Redes de Computadoras

3. Mostrar todos los cursos

4. Consultas básicas (obligatorias)

Resuelve las siguientes:

a) Mostrar solo los cursos donde el área sea 'Computación'
b) Mostrar los cursos que tengan más de 5 créditos
c) Mostrar los nombres de los cursos ordenados alfabéticamente
d) Mostrar solo 3 cursos usando LIMIT

(Ej. los primeros 3)

5. Actualizar un registro

Actualiza los créditos de uno de tus cursos, ejemplo:

Cambia los créditos de "Programación I" a 7.

6. Eliminar un registro

Elimina alguno que no necesites, por ejemplo:

Eliminar "Literatura Universal" de la tabla.

7. Consulta de verificación

Después de eliminar, muestra nuevamente todos los registros para confirmar que la operación fue correcta.
